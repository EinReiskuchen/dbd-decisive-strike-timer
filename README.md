# Dead By Daylight Decisive Strike Timer - 黎明杀机打击计时器

## 项目简介
这是一个为windows平台《黎明杀机》（Dead by Daylight）游戏设计的计时器工具，专门为屠夫玩家提供“果断反击/毁灭打击”技能的60秒倒计时提醒。

## 使用说明
- **默认热键**: 
    - 鼠标下侧键启动计时
    - 上侧键清空所有计时器。
- **准星**:
    - 默认在屏幕正中间显示，绿色十字样式。
    - 右键菜单：可“显示/隐藏”
- **窗口操作**:
    - 左键拖动窗口。
- **托盘图标**:
    - 左键单击：显示/隐藏窗口。
    - 右键菜单：提供“显示/隐藏”和“退出”选项。

## 技术栈
- **核心语言**: Python 3.8+
- **GUI框架**: tkinter
- **输入监听**: pynput
- **系统托盘**: pystray
- **音效处理**: winsound
