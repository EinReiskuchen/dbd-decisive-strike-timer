# Dead By Daylight Decisive Strike Timer - 黎明杀机打击计时器

## 项目简介
这是一个为windows平台《黎明杀机》（Dead by Daylight）游戏设计的计时器工具，专门为屠夫玩家提供“果断反击/毁灭打击”技能的60秒倒计时提醒。

## 安装说明
1. 克隆项目到本地。
2. 在虚拟环境中安装依赖：`pip install -r requirements.txt`
3. 运行程序：`python -m src.main`

## 使用说明
- **默认热键**: 
    - 鼠标下侧键启动计时
    - 上侧键清空所有计时器。
- **窗口操作**:
    - 左键拖动窗口。
    - 点击右上角 "✕" 按钮可将窗口隐藏到系统托盘。
- **托盘图标**:
    - 左键单击：显示/隐藏窗口。
    - 右键菜单：提供“显示/隐藏”和“退出”选项。

## 开发计划
- **按键自定义**：设置哪个按键来启动或清除。
- **声音自定义**：设置倒计时的提醒音效。

## 技术栈
- **核心语言**: Python 3.8+
- **GUI框架**: tkinter
- **输入监听**: pynput
- **系统托盘**: pystray
- **音效处理**: winsound

## 声明
该软件完全是玩家自行操作的倒计时以及提醒工具，目前不涉及任何自动化的部分

## Release版本
1.0版本完全是基于控制台的音效提醒版本，不包含任何GUI
